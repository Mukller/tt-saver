# TT Saver

Телеграм бот для скачивания видео из TikTok

## Возможности

- Скачивание видео из TikTok по ссылке
- - Поддержка yt-dlp для загрузки
  - - Интеграция с Telegram через WebApp
    - - Обработка ограничений TikTok
     
      - ## Требования
     
      - - Python 3.8+
        - - yt-dlp
          - - FFmpeg
            - - Telegram Bot API
             
              - ## Установка
             
              - ```bash
                git clone https://github.com/Mukller/tt-saver.git
                cd tt-saver
                pip install -r requirements.txt
                ```

                ## Использование

                1. Отправьте ссылку на видео TikTok боту
                2. 2. Бот скачает видео и отправит вам
