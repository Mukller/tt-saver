# Release Information

## Version 1.0.0 - 2026-05-20

### Release Type: Initial Release

This is the first stable release of the TikTok Telegram Bot.

### What's Included

- Core bot functionality for TikTok video downloading
- Full async/await support for high concurrency
- Integration with Telegram Bot API
- Robust video processing with yt-dlp
- Comprehensive error handling
- Systemd service configuration
- Docker support

### System Requirements

- Python 3.10 or higher
- pip package manager
- Modern Telegram client
- Internet connection

### Known Limitations

- Rate limiting may apply from TikTok
- Some videos may not be downloadable due to region restrictions
- Large videos may take time to process
